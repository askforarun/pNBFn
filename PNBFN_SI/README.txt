Execute the following command from PNBFN_SI folder to build a polymer. 
./scripts/create_polymersystems.sh

The script will create five folders, PNBF0, PNBF4, PNBF6, PNBF8, PNBF10. Each folder will contain 3 replicas. After running the script, the ouput will be PNBbox.gro. Please note modef suite must be installed before running this script. 
The recipe for building the polymer chain, PNBFN.py is in /PNBFN_SI/mol2_ff_files
